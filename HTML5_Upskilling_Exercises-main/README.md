# HTML5 Upskilling Exercises

This repository contains solutions for the HTML5 exercises completed as part of the **Cognizant Digital Nurture 5.0 Upskilling Program**.

## Project Theme

**Local Community Event Portal**

The project demonstrates the use of HTML5, CSS, JavaScript, Web Storage APIs, Geolocation API, Media Elements, and Browser Event Handling.

## Exercises Included

### 1. Create the HTML5 Base Template

* HTML5 document structure
* Semantic elements
* Comments and metadata

### 2. Navigation and Linking

* Navigation menu using `<nav>`
* Internal page links
* External links

### 3. Welcome Message with Styling and ID/Class

* Internal CSS
* ID and Class selectors
* Inline styling

### 4. Image Gallery for Community Events

* Tables and images
* Image attributes (`alt`, `title`)
* CSS styling

### 5. Event Registration Form

* HTML5 form elements
* Validation attributes
* Output element

### 6. Event Feedback with Event Handling

* onblur
* onchange
* onclick
* ondblclick
* Keyboard events

### 7. Video Invite with Media Events

* HTML5 video element
* oncanplay event
* onbeforeunload event

### 8. Saving User Preferences

* localStorage
* sessionStorage
* Clearing stored data

### 9. Geolocation for Event Mapping

* Geolocation API
* Position retrieval
* Error handling

### 10. Debugging with Chrome DevTools

* Console logging
* Debugging techniques
* Browser Developer Tools
